My resume in LaTeX.

All credit to the original author [Scott Clark](https://github.com/sc932).

In linux run
```bash
$ pdflatex resumeName.tex
```
This should result in the creation of ``resumeName.pdf``

Ubuntu requirements:
```bash
$ sudo apt-get install texlive texlive-latex-extra
```
